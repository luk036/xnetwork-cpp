template<typename _Graph>
py::GrAdaptor class

Graph adaptor for Boost Graph Library interoperability.

Template parameters
_Graph The Boost graph type

Contents

  • Reference
    • Base classes
    • Public static functions
    • Constructors, destructors, conversion operators
    • Public functions

This class provides a bridge between xnetwork and Boost Graph Library, allowing Boost algorithms to work with xnetwork graph structures.
